# APK Studio

> A powerful cross-platform IDE for Android APK analysis, editing, debugging and project management.

APK Studio is a modern desktop application designed for Android developers who need an advanced environment for inspecting, analyzing, modifying, rebuilding, and managing Android APK projects.

Built with Electron, React, TypeScript, and Monaco Editor, APK Studio combines the familiarity of modern development environments with powerful Android-specific tooling.

---

## Features

### Workspace System

* APK Studio Project (`.asproj`) workspaces
* Multiple workspaces support
* Auto-save and crash recovery
* Local history and backups
* Project templates

### Modern Editor

* Monaco Editor integration
* Multi-tab editing
* Syntax highlighting
* File explorer
* Search and navigation tools

### Android Development Tools

* APK analysis
* Manifest inspection
* Resource explorer
* APK rebuilding
* APK signing management
* Multiple Android toolchain support

### Device Tools

* ADB integration
* Device manager
* Logcat viewer
* Device explorer

### Visual Editors

* Manifest editor
* Theme editor
* Resource editor
* Layout editor

### Security Analysis

* Permission analysis
* Certificate inspection
* APK structure analysis
* Security reporting

### AI Assistant

Supports multiple providers:

* Google Gemini
* OpenAI
* Ollama (local models)

Features:

* Code explanations
* Resource analysis
* Manifest analysis
* Development assistance

### Plugin System

* Extensible architecture
* Plugin marketplace *(planned)*
* Custom panels
* Custom analyzers
* Language packs
* Themes

### Internationalization

* Runtime language switching
* Community language packs
* RTL/LTR support
* Multiple built-in languages
